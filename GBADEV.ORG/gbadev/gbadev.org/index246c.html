3D Engine: Quake GBA homage
By krom [Sunday October 31st 03:11:53 UTC 2004]		 News : General
Permalink

A good 3d engine has been released by Tomer Shalev here is an excerpt from the readme.txt:

Well, I decided to contribute a litle to the community and release this 3D demo.
This demo was done as part of my 3D programming learning for the past year, and for some reason I fell in love with GBA development, so I decided to do it as GBA project. This engine will be called : The Quake GBA homage, as it was based upon other people knowledge, so this is my contribution.
The level featuring in the demo was made using a Quake editor. This engine can run all Quake 1 levels I'll probabely release it open-source, but that depends on demand.
Would like to thank you all for knowledge pouring in the forums, and to Yeti3D, which was a real motivation for me to code 3D software.

Find out more about the features of the 3D engine from the readme.txt file.

Grab the 3D engine demo straight from this link: 3d2.zip [968kb]
