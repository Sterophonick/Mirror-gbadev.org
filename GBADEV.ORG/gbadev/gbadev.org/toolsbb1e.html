higan v105 [Tools/Emulator]

Made by: byuu
Released: 2017-10-08
Link: https://byuu.org/emulation/higan/

higan is a multi-system emulator that began development on 2014-10-14.

It currently supports the following systems:
• Nintendo Famicom
• Nintendo Super Famicom
• Nintendo Game Boy
• Nintendo Game Boy Color
• Nintendo Game Boy Advance
• Sega Master System
• Sega Game Gear
• Sega Mega Drive
• NEC PC Engine
• NEC SuperGrafx
• Bandai WonderSwan
• Bandai WonderSwan Color

higan also supports the following Super Famicom subsystems:
• Super Game Boy
• BS-X Satellaview
• Sufami Turbo

Minimum System Requirements:
• Intel Core-series processor
• OpenGL 3.2-capable graphics card
• Windows XP+, OS X 10.7+, Linux 3.2+ or FreeBSD 10.0+

This release provides several major improvements to Mega Drive emulation which enhances compatibility a good deal. It also includes important Super Famicom mosaic emulation improvements, plus a much-needed SuperFX save state issue fix.

This will be the last release for a while: I have an upcoming vacation, and following that, I'll be shifting my focus back to the SNES preservation project until my backlog of 125 PAL games is cleared.
