Simple Basketball Demo for GBA [Demos/Tech Demo]

Made by: 3DSage
Released: 2016-02-07
Download: SimpleBasketballDemoForGBA.zip


Hello GBA developers! I'm glad to see people are still interested in such a great handheld system.

I have been trying to teach myself how to make a game for the GBA so I thought I would put together a very basic demo with lots of comments for a beginner to learn from.

This demo is honestly not fun to play, but the C source code covers topics that you will need to know if you want to make a simple game. This demo will show you mode 4 double buffering, how to draw and move an image, fast DMA background, key inputs, count seconds, print numbers, show frames per second, and play sound. All this in around 125 lines of code with a batch file to compile.

Good luck to you and I hope you enjoy programming on the GBA as much as I do!
