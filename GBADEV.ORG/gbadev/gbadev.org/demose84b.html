Triple Triad [Games/Fighting]

Made by: Hugh Lowry
Released: 2005-04-02
Download: TripeTriad.zip


The current build features an intro, menu system and two short levels,
in addition to a tutorial level (all selectable via the 'level select'
option). It works on both the GBA hardware and on all tested GBA
emulators. The code is about 10,000 lines in total (.h and .cpp files),
with sound effects and graphics on top of that. The compiled file is
currently 3.28mb.

Triple Triad has been in development since September 2004, and
is due to be assesed in March 2005, although I would like to
continue work on it for a bit after that. The game was developed from
scratch using C/C++ and the DevkitAdvance compiler. I have tried
to use my original artwork wherever possible, although some elements
have been taken from other games - specifically the character
outlines/animations are taken from Capcom's Megaman games, as are
some of the background tiles. The engine as it stands is complete,
but I will be constantly tweaking the enemy AI and player moves until
the project deadline (March 18th)

Features not yet implemented:
*The options entry in the menu currently does nothing - it just loads
a new game if selected
*The player can run on the road in level 2. They should lost a heart
of energy - I haven't implemented this yet as it makes playtesting the
whole level a bit tricky.
*The laser beams on the mech enemy don't do any damage currently. Same
reason as above.

Known issues:
*Very occasionally the player gets stuck in a loop of repeatedly being
hit, even if no enemies are nearby.
*Enemies sometimes stand offsceen to fire at the enemy - you just
have to wait for them to come back on screen
*The game may freeze up after a while if music is enabled. It does the
same if only sound effects play, but takes longer to do so. Looks like
a memory/stack problem, but I haven't been able to sort it.
